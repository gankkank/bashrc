Bashrc
------

### Contents

- [x] Command alias for: git, ls, bundle, rspec
- [x] Custom PS1.
- [x] SSH agent.

### Install

`git clone https://github.com/gankkank/bashrc ~/.bash`

`echo '. ~/.bash/init.sh' >> ~/.bash_profile`

### Setup 

`curl -L https://github.com/gankkank/bashrc/raw/master/setup.sh 2> /dev/null | bash; exit`
